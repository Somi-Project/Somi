# executive

Governance, planning, proactivity, and prompt assembly layer.

## Responsibilities
- Approval and execution governance (`istari*`, queue, approvals).
- Strategic planning and synthesis.
- Memory orchestration helpers.
- Prompt construction (`promptforge.py`, `prompting/`).

## Integration points
- Routing decisions come from `workshop.toolbox.agent_core.routing`.
- Contract/artifact primitives come from `workshop.toolbox.stacks.contracts_core`.
