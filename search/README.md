# search

Search-time memory and retrieval helpers.

This package is smaller than the web/research stacks. It focuses on session and
memory retrieval that support continuity rather than open-web browsing itself.

## Key Files

- `session_search.py`
  - search across persisted session history and memory-oriented records

## Read This Package When

- you are debugging memory-backed search or session recall
- a request seems to pull from prior conversations instead of the open web
- you want the bridge between runtime memory and search behavior
