# config

Central configuration for models, routing, tools, integrations, and runtime behavior.

## Contents
- `settings.py`: primary runtime toggles.
- `*_settings.py`: domain-specific settings (heartbeat, toolbox, skills, twitter, memory, search).
- `personalC.json`: persona catalog.
- `assistant_profile.json`: active profile state.
- `extraction_schemas/`: OCR extraction schema files.
