# workshop/skills

Skill parsing, eligibility, and dispatch orchestration.

## What this folder does
- Parses skill metadata and invocation intent.
- Applies skill safety/eligibility gating.
- Routes skill execution through toolbox bridge/runtime paths.
- Records recent skill runs for operator visibility.
