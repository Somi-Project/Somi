# SOMI v0.92 - Mineralization

Mineralization is the release where SOMI stops feeling like a promising framework and starts feeling like a real local AI operating system.

This version is about hardening, densifying, and refining the core. More capability. Better structure. Stronger control. Cleaner operator experience.

Under pressure, SOMI did not crack. It crystallized.

---

## Release Theme

`v0.92 Mineralization` is a foundation release.

It takes SOMI's major systems and turns them into something more durable:

- more modular
- more secure
- more observable
- more self-hosted
- more usable for real people

This is the chapter where the framework becomes far more credible as a daily driver for local AI.

---

## Highlights

### SOMI Now Feels Like A Real AI Workstation

SOMI now delivers a much more complete local operator experience with:

- Control Room
- Coding Studio
- Research Studio
- Speech controls
- Node Manager

This is no longer just a command loop with extra tools attached. It is a structured desktop-first environment for running your own AI system.

### Coding Mode Became Serious

SOMI now includes a much stronger local coding stack with:

- managed coding workspaces
- guarded file operations
- Python-first coding workflows
- multi-language workspace support
- verify loops and scorecards
- a dedicated coding cockpit

The goal was simple: make local coding in SOMI feel closer to a serious coding agent instead of a novelty feature.

### Research Got Much Stronger

Research mode now goes beyond plain search-and-summarize.

SOMI now supports:

- research supermode
- evidence graphs
- grounded exports
- contradiction-aware synthesis
- longer-lived research workflows

This release makes SOMI much more competitive as a self-hosted research system.

### Speech Was Rebuilt

The speech layer was overhauled into a cleaner, more practical local-first pipeline.

SOMI now uses:

- local TTS via `pyttsx3`
- local STT via `faster-whisper`
- diagnostics and health checks
- desktop speech controls

The result is a speech system that is easier to understand, easier to install, and much more believable for everyday use.

### Security And Control Matured

Mineralization pushed hard on trust and operational safety.

Major gains include:

- approval-aware execution
- scoped remote behavior
- gateway and node mesh foundations
- runtime audit trails
- release gate and framework freeze tooling
- stronger control-plane visibility

SOMI is built to be powerful without pretending risk does not exist.

### The Backbone Is Stronger

A huge amount of work in this release went into the parts most users never see directly, but feel immediately:

- split monoliths
- cleaner package boundaries
- workflow runtime
- subagent runtime
- ontology-backed actions
- persistent state plane
- memory search and compaction resilience
- tool registry and execution backends
- skill forge and marketplace foundations

This is the kind of work that makes a framework survive growth.
